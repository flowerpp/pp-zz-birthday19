## Phase 0 研究

无需额外研究。技术决策继承 002 已确立的模式：

- 署名段落位于 `index.html` 的 `#intro` 区 `<button id="start-btn">` 之后
- 样式规则在 `css/scenes.css` 的 `.intro-credits` 中已定义（`word-break: break-word`，小屏自动换行）
- 时间分隔采用 `&emsp;`（全角空格）与 `<br>` 结合排版，与 002 模式一致

**已实现的 HTML 结构**：

```html
<p class="intro-credits intro-line">
  软件作者：冯存华、邬巧玲<br>
  开始开发：2026-03-10 23:59:59&emsp;最后更新：2026-03-11 11:36:36
</p>
```

## Phase 1 设计

无新增数据实体、无 API 契约、无外部接口，跳过 data-model.md 和 contracts/。

**时间戳维护约定**（操作规范）：
- `开始开发` 值永久固定为 `2026-03-10 23:59:59`，不得修改
- `最后更新` 值在每次修改 `index.html` 后由开发者手动更新为当前本地时间，格式 `YYYY-MM-DD HH:MM:SS`
- 该更新动作属于开发者手工维护，不引入任何脚本或构建步骤
