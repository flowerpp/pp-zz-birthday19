# 实现规划：《给19岁的你》— 壮壮生日互动叙事网页游戏

**分支**: `001-birthday-memory-game` | **日期**: 2026-03-11 | **规格**: [spec.md](./spec.md)
**输入**: 功能规格来自 `/specs/001-birthday-memory-game/spec.md`

---

## 摘要

为壮壮19岁生日制作一个纯静态互动叙事网页游戏，通过4个章节（序幕→8幕记忆长廊→19朵星光→终幕彩蛋）呈现19年成长历程。
技术方案：原生 HTML5 + CSS3 + 纯 JavaScript，无后端依赖，照片压缩后内嵌，部署至 GitHub Pages。

---

## 技术背景

**语言/版本**: HTML5 + CSS3 + JavaScript ES2020+
**主要依赖**: 无第三方框架（纯原生实现）；GSAP 3.x（动画增强，可选）
**存储**: N/A — 无持久化存储，所有资源静态内嵌
**测试**: 手动测试（iOS Safari 17+、Android Chrome 120+、桌面 Chrome/Safari）
**目标平台**: 手机浏览器首选（iOS Safari、Android Chrome）；PC 浏览器兼容
**项目类型**: 纯静态网页 / 互动视觉叙事体验
**性能目标**:
  - 首屏加载 ≤ 5 秒（手机 4G 网络）
  - 场景切换动画流畅（≥ 30fps）
  - 总资源体积 ≤ 15MB（压缩后图片 + 音频）
**约束**:
  - 无服务器、无数据库、无 CDN 依赖
  - GitHub Pages 兼容（纯静态文件）
  - 所有照片压缩至 WebP 格式，单张 ≤ 300KB
  - 音频文件 mp3 格式，≤ 2MB
**规模/范围**: 单用户访问，无并发压力；8个场景 + 19颗星 + 1个彩蛋

## 复杂度追踪

*仅记录宪法检查中需要豁免说明的项*

| 豁免 | 为什么需要 | 更简单替代方案被拒绝原因 |
|------|-----------|----------------------|
| 跳过自动化测试 | 纯 CSS 动画 + Canvas 粒子效果无法被单元测试验证效果 | 手动验收清单可覆盖所有验收场景，自动化测试无额外价值 |
| 无构建工具 | 减少环境依赖，降低部署复杂度，方便家长后续维护 | Vite/Webpack 引入工具链复杂度，对此静态项目过度工程化 |
